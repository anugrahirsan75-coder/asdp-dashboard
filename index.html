⚠ Konfigurasi API Belum Diisi

Variabel API_BASE di dalam file index.html masih kosong atau belum benar.

Cara memperbaiki:

1. Buka Apps Script Editor → Deploy → Manage Deployments → copy URL Web App (berakhiran /exec).

2. Edit file index.html, cari baris var API_BASE = '…', ganti dengan URL yang sudah di-copy.

3. Commit dan push ke GitHub. Tunggu ±1 menit, lalu refresh halaman.
